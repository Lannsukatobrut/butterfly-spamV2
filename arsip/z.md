### Cara Install
<details open>

```php
$ pkg update && pkg upgrade
$ pkg install python
$ pkg install git
$ git clone https://github.com/Madd-KW/butterfly-spamV2
$ cd butterfly-spamV2
$ pip install -r requirements.txt
$ python run.py

Untuk update script nya cukup ketik perintah
$ cd butterfly-spamV2
$ git pull
$ ls
$ python run.py
```
