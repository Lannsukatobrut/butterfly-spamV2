### Cara Install
<details open>

```php
$ pkg update && pkg upgrade
$ pkg install python
$ pkg install git
$ git clone https://github.com/Madd-KW/butterfly-spamV2
$ cd butterfly-spam
$ make setup
$ python run.py

Atau untuk update script nya dg ketik perintah
$ cd butterfly-spam
$ git pull
$ ls
$ python run.py
```
</details>
